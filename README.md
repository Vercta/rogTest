你来到了“rogTest”的发布页面：**rogTest** 旨在实现[Unity中文课堂《王国之梦》教程](https://learn.u3d.cn/tutorial/CardCourse-DOTK)中的所有功能，最终用于提交 “作业一” 的工程任务(8.23)

## 关于本项目
这是一款基于 Unity 2D 开发的Roguelike卡牌游戏，目前内容并未实现完整。
